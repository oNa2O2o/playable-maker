# Playable Maker - 试玩广告批量生成工具

自动化试玩广告制作流程：扫描视频素材 → 识别角色特征 → 生成多语言多渠道广告包。基于 playable-editor 工具，支持 Google、TikTok 等主流投放平台。

## When to Apply

使用此 skill 当：
- 用户要制作试玩广告（playable ads）
- 用户提供了视频素材需要批量生成
- 用户要生成多语言版本的试玩广告
- 用户提到"做试玩"、"playable"、"互动广告"

## 核心原则

### 原则1：视觉优先命名
- **为什么**：角色名称需要跨语言识别，视觉特征（发色、服装、气质）是最稳定的标识
- **什么时候**：为每个视频素材命名时，使用"发色+特征+气质"格式
- **反例**：不要用"女性向1"、"素材A"等无意义标签

### 原则2：性向明确分类
- **为什么**：投放时需要按目标受众分类，文件名必须包含性向标识
- **什么时候**：生成文件名时，在渠道后添加性向（女性向/男性向）
- **反例**：不要混淆性向，不要省略性向标识

### 原则3：平台适配压缩
- **为什么**：不同平台对文件大小有不同限制，TikTok 要求更严格
- **什么时候**：TikTok 渠道使用额外压缩（1500KB），Google 使用标准压缩（2100KB）
- **反例**：不要所有渠道用同一压缩率

### 原则4：语言本地化
- **为什么**：不同市场需要不同语言版本，角色名也要本地化
- **什么时候**：为每个角色准备 TC（繁中）、JP（日文）、EN（英文）三个版本的名字
- **反例**：不要只用拼音或英文名

## 工作流

### 阶段一：素材扫描与角色识别

**输入确认**：
- 素材目录路径
- 参考成品路径（可选）
- 对话文案（如果有）

**视觉分析**：
- 提取视频第一帧作为预览图
- 识别角色特征：发色、服装、表情、气质
- 判断性向：根据角色外观和用户说明

**角色命名**：
- 视觉标签：如"黑发西装冷峻"、"粉发甜美少女"
- 本地化名字：
  - TC：符合繁体中文习惯的名字
  - JP：日文名字（含假名注音）
  - EN：英文名字

### 阶段二：配置生成

**语言版本**：
- 如果用户提供了对话文案，使用用户的文案
- 如果没有，使用默认模板
- 确保 TC、JP、EN 三个版本的对话内容一致

**渠道配置**：
- GG（Google）：标准压缩
- TT（TikTok）：额外压缩 + config.json

**文件命名规则**：
```
MM_{渠道}_{性向}_{角色视觉标签}_{语言}_{日期}.zip
```

### 阶段三：批量生成

**生成流程**：
1. 复制视频到 playable-editor 的 uploads 目录
2. 压缩视频（根据渠道选择压缩率）
3. 为每个组合生成 HTML
4. 打包成 ZIP 文件
5. 输出统计报告

**编码处理**：
- Windows 环境下修复 UTF-8 编码输出
- 确保中文文件名正确显示

**输出验证**：
- 检查文件大小是否符合平台要求
- 统计生成的文件数量
- 生成说明文档

### 阶段四：交付

**输出内容**：
- 所有 ZIP 广告包
- 生成说明文档（包含角色列表、命名规则、使用说明）
- 视频预览图（用于确认角色）

**文件组织**：
```
输出目录/
├── MM_GG_女性向_黑发西装冷峻_TC_20260518.zip
├── MM_GG_女性向_黑发西装冷峻_JP_20260518.zip
├── MM_GG_女性向_黑发西装冷峻_EN_20260518.zip
├── MM_TT_女性向_黑发西装冷峻_TC_20260518.zip
├── ...
├── MM_GG_男性向_粉发甜美少女_TC_20260518.zip
├── ...
└── 生成说明.md
```

## 技术规范

### 工具依赖
- playable-editor：位于 `D:\Claude Code\playable-editor`
- Python 脚本：`batch_export_*.py`
- opencv-python：用于视频帧提取

### 视频要求
- 格式：MP4
- 比例：竖屏 9:16（推荐 1080x1920）
- 时长：5-15秒
- 大小：原始文件 < 10MB

### 压缩目标
- Google (GG)：2100KB（约 2MB）
- TikTok (TT)：1500KB（约 1.5MB）

### 文件结构
```python
VIDEOS = [
    {
        "path": "素材路径",
        "tag": "角色视觉标签",
        "gender": "性向",
        "name_TC": "繁中名",
        "name_JP": "日文名",
        "name_EN": "英文名",
    },
]
```
