# Playable Ad Editor 操作指南

> 本文档面向实习生，帮助你快速上手：**替换文案、更换背景视频、导出广告包**。

---

## 一、启动工具

双击 `PlayableEditor.exe`，黑色窗口出现后浏览器会自动打开编辑器页面。

> **注意**：黑色窗口不要关闭，它是后台服务。关闭它 = 关闭工具。

打开后你会看到这个界面：

```

│   📋 基础配置            │                                          │
```

## 二、替换对话文案

### 2.1 直接在列表中编辑（最常用）

找到左侧 **「💬 对话消息」** 区域，你会看到消息列表：

```
┌──────────────────────────────────────────────────┐
│ ⠿  AI   [ Are you awake?          ]  500ms  ✏ ✕ │
│ ⠿  USR  [等待发送]                          ✏ ✕ │
│ ⠿  AI   [ Can I ask you something? ] 700ms  ✏ ✕ │
│ ⠿  END  [结束触发]                          ✏ ✕ │
│                                                  │
│ [+ AI消息]  [+ 用户触发]  [+ 结束触发]           │
└──────────────────────────────────────────────────┘
```

- **蓝色 AI 标签** 后面是可编辑的输入框，直接点击修改文字即可
- 修改后右侧预览会**自动更新**，无需手动保存

**操作步骤**：
1. 点击 AI 行的文本框
2. 清空原文，输入新文案
3. 右侧预览自动刷新 → 确认效果

### 2.2 修改消息类型或延迟时间

点击消息行右侧的 **✏ 按钮**，弹出编辑窗口：

```
┌────────────────────────────────┐
│  编辑消息                      │
│                                │
│  类型：[AI 消息         ▼]     │
│                                │
│  消息内容：                    │
│  ┌────────────────────────┐   │
│  │ Hey, are you there?    │   │
│  └────────────────────────┘   │
│                                │
│  前一条消息后延迟 (ms)：500     │
│                                │
│              [取消]  [保存]    │
└────────────────────────────────┘
```

- **类型**：一般不需要改
- **延迟**：控制这条消息出现前等待的毫秒数（500 = 半秒）

### 2.3 添加 / 删除消息

- **添加**：点击列表下方的 `[+ AI消息]` 按钮，自动追加一条新消息
- **删除**：点击消息行右侧的 **✕ 按钮**
- **排序**：按住消息行左侧的 **⠿ 拖拽手柄**，上下拖动调整顺序

### 2.4 修改结束页文案

找到 **「🏁 结束页」** 区域：

```
┌──────────────────────────────────┐
│  标题：    [ Discover Your AI Soulmate ]   │
│  描述：    ┌──────────────────────┐        │
│            │ Start meaningful ... │        │
│            └──────────────────────┘        │
│  CTA按钮文案：[ Download ]                  │
└──────────────────────────────────┘
```

直接修改即可，预览自动同步。

## 三、更换背景视频

找到左侧 **「🎨 资源管理」** 区域：

```
┌──────────────────────────────────────────────────┐
│  Logo URL:                                        │
│  [ https://xxx/LOGO2.png                ] [上传]  │
│                                                   │
│  背景视频 URL:                                     │
│  [ https://xxx/ComfyUI_00077.mp4        ] [上传]  │
└──────────────────────────────────────────────────┘
```

### 方式一：上传本地文件（推荐）

1. 点击 **「上传」** 按钮
2. 选择本地的 `.mp4` 视频文件
3. 上传成功后，URL 自动替换，右侧预览自动刷新

### 方式二：粘贴在线链接

1. 清空「背景视频 URL」输入框
2. 粘贴新的视频链接（必须是直链，以 `.mp4` 结尾）
3. 预览自动刷新

> **视频要求**：竖屏 9:16 比例最佳，建议 720x1280 或 1080x1920，文件大小控制在 5MB 以内。

## 四、保存配置

修改完成后，**必须保存配置**，否则刷新页面会丢失。

### 保存到当前配置

点击顶部栏的 **[保存]** 按钮。

### 另存为新配置

1. 点击顶部栏的 **[另存为]** 按钮
2. 输入配置名称（如 `Eric_版本2`）
3. 确认

### 切换配置

点击顶部栏的下拉菜单，选择之前保存的配置即可切换。

```
┌─────────────────────────────────────────────────────────┐
│  [-- 新建配置 --  ▼]  [保存]  [另存为]  [删除]          │
│  ├ Eric_版本1 (MiraiMind, 4条)                          │
│  ├ Eric_版本2 (MiraiMind, 6条)                          │
│  └ Sakura_测试 (LoveApp, 4条)                           │
└─────────────────────────────────────────────────────────┘
```

## 五、预览效果

修改内容后，右侧手机预览会自动更新画面。

如需**完整播放一遍**对话流程：
1. 点击预览下方 **[▶ 播放预览]** 按钮
2. 手机画面中会依次弹出 AI 消息
3. 出现键盘时，随便输入文字点发送，继续对话
4. 最后显示结束页

如需重头再来，点击 **[↻ 重置]**。

## 六、导出广告包

找到左侧最下方 **「📦 导出」** 区域：

```
┌──────────────────────────────────────────┐
│  [  TikTok  ] [  Google  ] [ Facebook ]  │
│                                          │
│  [        一键导出全部平台        ]       │
│                                          │
│  本地化导出（勾选后额外生成翻译版本）      │
│  □ 日本語 (JP)  □ English (EN)  □ 繁體中文 (TC) │
│                                          │
│  ✓ tiktok: 120.5 KB — 下载链接           │
│  ✓ google: 118.2 KB — 下载链接           │
│  ✓ facebook: 119.8 KB — 下载链接         │
└──────────────────────────────────────────┘
```

### 6.1 导出单个平台

点击 **TikTok / Google / Facebook** 按钮，自动生成该平台的 ZIP 包并下载。

### 6.2 一键导出全部

点击 **[一键导出全部平台]**，自动生成 3 个平台的 ZIP 包并下载。

### 6.3 导出多语言版本

1. 在导出按钮下方，**勾选**需要的语言：JP（日语）、EN（英语）、TC（繁体中文）
2. 再点击导出按钮
3. 工具会自动翻译文案，为每个语言生成额外的 ZIP 包

例如勾选了 JP 并导出全部平台，会生成 **6 个文件**：
```
MiraiMind_Playable_Tiktok_20260306.zip       ← 原始中文版
MiraiMind_Playable_Google_20260306.zip
MiraiMind_Playable_Facebook_20260306.zip
MiraiMind_Playable_Tiktok_JP_20260306.zip    ← 日语版
MiraiMind_Playable_Google_JP_20260306.zip
MiraiMind_Playable_Facebook_JP_20260306.zip
```

## 七、导出文件在哪里？

导出的 ZIP 文件会**自动触发浏览器下载**，默认保存在你电脑的「下载」文件夹中。

### 命名规则

```
{应用名}_Playable_{平台}_{语言}_{日期}.zip
```

示例：`MiraiMind_Playable_Tiktok_JP_20260306.zip`

### 发给投放同事

直接把下载到的 `.zip` 文件发给对应平台的投放同事即可，**不需要解压**。

## 八、完整操作流程总结

以一次典型的「替换文案 + 换背景 + 导出」任务为例：

```
1. 双击 PlayableEditor.exe 启动
         ↓
2. 顶部选择已有配置（或新建）
         ↓
3. 「🎨 资源管理」→ 点击"上传"替换背景视频
         ↓
4. 「💬 对话消息」→ 直接修改每条 AI 消息的文案
         ↓
5. 「🏁 结束页」→ 修改标题、描述、按钮文案
         ↓
6.  右侧预览确认效果 → 点击 [▶ 播放预览] 完整看一遍
         ↓
7.  顶部点击 [保存] 保存配置
         ↓
8. 「📦 导出」→ 勾选需要的语言 → 点击 [一键导出全部平台]
         ↓
9.  浏览器自动下载 ZIP 文件 → 发给投放同事
```

## 九、常见问题

**Q: 修改后预览没变化？**
A: 点击 [↻ 重置] 再点 [▶ 播放预览]。

**Q: 上传视频很慢？**
A: 视频文件过大。建议先用其他工具压缩到 5MB 以内。

**Q: 导出时显示"导出中..."很久？**
A: 如果勾选了多语言，需要调用翻译接口，请耐心等待（通常 10-30 秒）。

**Q: 关掉黑色窗口后打不开页面了？**
A: 重新双击 `PlayableEditor.exe` 即可。

**Q: 提示"发现新版本"？**
A: 点击顶部 [检查更新] 按钮，按提示更新即可。
